From `docs/DOCKER-BUILD-STRATEGIES.md`.

- Development: docker-compose.yml (hot reload)
- Web-only or API-only compose files for partial dev
- Production:
  - Build-time Dockerfile (faster startup, larger image)
  - Runtime Dockerfile (slower startup, flexible)
